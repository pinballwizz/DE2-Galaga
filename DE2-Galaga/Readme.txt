Galaga Arcade for the Altera DE2-35 Dev Board.

Notes:
Controls by PS2 keyboard see original readme files in the de2 folder.
t = credit
f = start
left/right = arrow keys
space = shoot

Build:
* Obtain correct roms file for galaga see script in tools/roms folder for rom filenames.
* Unzip rom files to tools/roms folder.
* Run the make galaga proms script in the tools/roms folder.
* Copy the generated prom files to the proms folder.
* Open the ckong_de2 project file using Quartus and compile.
* Program DE2 Board.
